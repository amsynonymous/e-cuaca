# 🌤 E-Cuaca

E-Cuaca is a modern weather web application that shows real-time weather information and a 7-day forecast for locations in Malaysia.

The app uses the Open-Meteo API and features a clean glassmorphism UI with animated weather backgrounds.

---

## 🚀 Features

- 🌤 Real-time weather data
- 📅 7-day weather forecast
- 🏙 Dropdown selection for Malaysian states and districts
- 🎨 Glassmorphism UI design
- ☁️ Animated weather background (sun, clouds, rain, stars)
- 📱 Fully responsive for desktop and mobile
- ⚡ Fast API integration using Open-Meteo

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Open-Meteo Weather API

---

## 📷 Preview

Weather information includes:
